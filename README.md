# Manufacturing Downtime Analysis  
### From Raw Sensor Data to Predictive Maintenance Insight

---

## Project Overview
Unplanned downtime is one of the most expensive challenges in manufacturing. It disrupts production, increases maintenance costs, and reduces equipment efficiency.

This project analyzes raw machine sensor data to:
- Understand downtime patterns  
- Identify failure-prone machines and assembly lines  
- Detect early warning signals from sensor behavior  
- Prepare data for predictive maintenance systems  

The approach combines **mechanical understanding** (pressure, temperature, vibration, torque, force) with **data analysis techniques** to transform raw data into actionable insights.

---

## Project Objective
- Clean and standardize inconsistent industrial data  
- Identify patterns associated with machine failure  
- Define safe operating ranges for equipment  
- Build a foundation for predictive maintenance models
